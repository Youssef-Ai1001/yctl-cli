# yctl-cli

## Overview

`yctl-cli` is a command-line interface for managing Youssef AI applications.

## Features

- Easy setup
- Configurable options
- Comprehensive logging

## Installation

To install, run:

```
npm install yctl-cli
```

## Usage

To start the CLI:

```bash
npx yctl-cli
```

### Commands

```bash
# List all commands
npx yctl-cli --help
```

### Examples

```bash
# Run a specific command
npx yctl-cli run
```
